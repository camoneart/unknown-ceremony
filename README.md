## CONCEPT

**"無名(私)が制作した作品をWebという大きな会場にて披露する式典"**

有名無名は関係ない。無名が存在するから有名も存在する。

無名な私が制作した作品もまた無名。それでもその作品一つ一つは私にとって "主役" であり、作品一つ一つに "式" を挙げてやりたい。そんな想いを込め、"UNKNOWN CEREMONY" と名付けました。

## ABOUT
